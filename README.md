# Server Performance Stats

Bu proje, bir Linux sunucusunun temel performans istatistiklerini analiz eden basit bir Bash scriptidir.

## Özellikler
- Toplam CPU, RAM ve Disk kullanımı.
- En çok kaynak tüketen 5 süreç.
- İşletim sistemi ve uptime bilgileri.

## Nasıl Çalıştırılır?
1. Dosyaya izin ver: `chmod +x server-stats.sh`
2. Çalıştır: `./server-stats.sh`
